Landing Page Layout

<br>

This short project is one of the first on _The Odin Project_'s _Foundations_ course, and involves using HTML and CSS to reproduce a static page layout that can be displayed in a browser. Uses basic html tags and elements, with flex boxes predominantly used to define the layout. It also uses placeholder image files with the HTML to act as placeholder image elements.

The layout works well on my monitor but does not adapt well to smaller screen sizes (1080p and below). page.png shows a screenshot of the page as displayed on my computer.
