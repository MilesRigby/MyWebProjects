Landing Page Layout

<br>

This short project is one of the first on _The Odin Project_'s _Foundations_ course, using HTML and CSS to reproduce a static page layout displayed in a browser. I used basic html tags and elements, with flex boxes as the main way of defining the layout. I also used placeholder image files provided by the course to fill image elements in the HTML.

The layout works well on my monitor but does not adapt well to smaller screen sizes (1080p and below). page.png shows a screenshot of the page as displayed on my computer.
