Admin Dashboard Layout

<br>

In this project, part of the _Intermediate HTML and CSS_ component of _The Odin Project_'s _Full Stack JavaScript_ course, I recreated a provided layout for a website admin page utlising CSS grid. The page has no functionality besides this, but represents an increase in complexity in terms of visual layout over the landing page project, as well as being my first project utilising CSS grid.

This project does not have responsive layout, and may not work well on most monitors. page.png shows how the layout appears on my personal monitor.
