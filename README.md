# README

This repository contains folders for many of the projects I have worked on while learning web development through _The Odin Project_. I have currently completed the _Foundations_ course, as well as the _Intermediate HTML and CSS_ and _JavaScript_ components of the _Full Stack JavaScript_ pathway. An overview of _The Odin Project_'s content is available on their website: https://www.theodinproject.com/paths

The goal of this repository is to document my progression in developing my skills with HTML, CSS, JavaScript, and tools such as webpack, Jest, and in the future React and NodeJS. It contains projects from throughout _The Odin Project_, so earlier projects are not representative of my current abilities.

Overview.txt provides an overview of the projects detailed in this repository in order of creation in a numbered list, providing links to both the repository for each project and a live web view if one is available. This is done using GitHub Pages.

Folders in this repository start with a number following the same order as Overview.txt, 01_, 02_, 03_, etc, followed by the project title. These folders contain an overview of each project, as well as some screenshots of them working on my personal computer. As I haven't yet learned to create fully responsive designs, quality may vary on screens of different resolutions than my monitor's 1440p.

My full public GitHub, including all projects completed as part of _The Odin Project_ on top of the ones listed in this repository, is available at: https://github.com/MilesRigby?tab=repositories
